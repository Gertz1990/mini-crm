# Mini CRM для фрилансеров

Небольшое приложение для управления списком клиентов. Позволяет добавлять, просматривать и удалять клиентов. Данные хранятся в `localStorage`, поэтому всё работает без сервера.

## Установка

```bash
cd mini-crm
npm install
```
В корне также установится серверный пакет `express`.

## Запуск в режиме разработки

```bash
npm run dev
```

Если хотите использовать backend API, запустите в другом терминале:

```bash
npm run start-server
```

Vite настраивает прокси для `/clients` на `http://localhost:3001`.

Приложение будет доступно на http://localhost:5173/.<br>
Внесённые изменения применяются моментально благодаря HMR.

## Сборка

```bash
npm run build
```

Содержимое будет в папке `dist`.

## Как работает

Форма вверху позволяет ввести имя, e-mail и телефон клиента. Клиенты отображаются списком с кнопкой «Удалить». Данные автоматически сохраняются в localStorage.

## Что можно добавить дальше

- Редактирование клиента
- Фильтрация/поиск
- Экспорт/импорт JSON или CSV
- Backend на Node/Express + база данных
- Тесты с React Testing Library

Это простая отправная точка для вашей мини-CRM.
