# Mini CRM Dashboard 📊

Современное приложение для управления клиентами с модульным **drag-and-drop дашбордом**.

## ✨ Возможности

- **Drag & Drop** - перетаскивайте панели для переупорядочения
- **Resizable** - меняйте размер панелей
- **Видимость модулей** - показывайте/скрывайте нужные панели
- **Автосохранение** - позиции и видимость сохраняются в localStorage
- **Статистика** - метрики по клиентам (всего, за месяц, популярные теги)
- **Полный CRUD** - добавление, редактирование, удаление клиентов
- **Поиск и фильтрация** - быстрый поиск по имени, email, телефону
- **Экспорт/Импорт** - JSON и CSV форматы
- **Apple-Style дизайн** - современный и красивый интерфейс

## 📋 4 Модульные панели

1. **📋 Форма добавления** - добавить новых клиентов
2. **🔍 Поиск и фильтры** - поиск, сортировка, экспорт/импорт
3. **📊 Статистика** - метрики и аналитика
4. **👥 Список клиентов** - таблица с редактированием

## 🚀 Быстрый старт

### Установка

```bash
npm install
```

### Разработка

```bash
npm run dev
```

Приложение запустится на http://localhost:5174/

### Production сборка

```bash
npm run build
```

Сборка создаст оптимизированный bundle в папке `dist`.

## 🌐 Развертывание на Koyeb

Приложение готово к развертыванию на Koyeb!

### Инструкции:

1. Синхронизируйте репозиторий на GitHub (уже готово ✓)
2. Создайте новый Koyeb сервис
3. Выберите GitHub репозиторий `mini-crm`
4. Koyeb автоматически найдет `Procfile`
5. Установите Environment variable:
   - `PORT` = 3001 (или другой)

**Автоматический процесс при деплое:**
- npm install (устанавливает зависимости)
- npm run build (собирает React приложение)
- node server/index.js (запускает Express сервер)

## 🤝 Как использовать

### Основные операции:

1. **Добавить клиента** - заполните форму в первой панели и нажмите "Добавить"
2. **Найти клиента** - используйте поле поиска во второй панели
3. **Редактировать** - нажмите "✏️ Изменить", отредактируйте данные, сохраните
4. **Удалить** - нажмите "🗑️ Удалить"
5. **Экспортировать** - нажмите "📥 JSON" или "📥 CSV"
6. **Импортировать** - нажмите "📤 Импорт" и выберите JSON файл

### Управление дашбордом:

- **Перемещение** - ухватитесь за заголовок (⋯) панели и тащите
- **Изменение размера** - потяните за нижний правый угол
- **Показать/Скрыть** - нажмите "☑️ Модули ▼" в заголовке
- **Сбросить макет** - нажмите "🔄 Сбросить макет" в панели статистики
